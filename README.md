## 📚 各教程详细介绍

### 1️⃣ 华尔街英语 —— 系统筑基篇

> 路径：`WallStreetEnglish/`
> 难度：**Pre-A1（零基础）** → **C1（高级精通）**
> 共 6 篇 28 单元 + 附录

基于华尔街英语 20 级课程体系，覆盖 CEFR 六个等级。每单元包含：核心词汇表（CEFR标注）、实用句型（中英对照）、语法要点、对话练习、自测题。

**适用对象**：英语基础薄弱或零基础，需要系统性搭建语法和词汇框架的学习者。

**学习建议**：按篇次顺序学习，每单元 1-2 小时。先学词汇 → 掌握语法 → 练习对话 → 完成自测。

**篇目结构**：

| 篇目 | CEFR | 级别范围 | 重点 |
|------|------|---------|------|
| 第一篇 · 入门篇 | Pre-A1 | Level 1-2 | 字母、问候、自我介绍 |
| 第二篇 · 基础篇 | A1 | Level 3-5 | 日常生活、现在时态、家庭 |
| 第三篇 · 初级进阶篇 | A2 | Level 6-9 | 购物旅行、过去时态、兴趣 |
| 第四篇 · 中级篇 | B1 | Level 10-13 | 职场沟通、社交、意见表达 |
| 第五篇 · 中高级篇 | B2 | Level 14-17 | 商务英语、讨论辩论、虚拟语气 |
| 第六篇 · 高级篇 | C1 | Level 18-20 | 专业写作、修辞、习语俚语 |

### 2️⃣ 走遍美国 —— 场景融入篇

> 路径：`FamilyAlbumUSA/`
> 难度：**A1** → **B2**

**适用对象**：有一定基础（A1+），希望通过真实生活场景学习地道表达的学习者。

| 第三篇·家庭与节日 | 感恩节、宠物、健康、生日 |
| 第四篇·生活与抉择 | 买房、蜜月、工作改变 |
| 第五篇·社区与情感 | 社区中心、跨代沟通、爱情 |

### 3️⃣ 老友记 —— 经典趣味篇

> 路径：`FriendsSeason/`
> 难度：**A2** → **B2**
> 共 6 篇 29 单元 + 附录

史上最受欢迎的情景喜剧之一（10 季 24 集），90% 对话由最常用的 2000 个单词构成。六位主角发音标准清晰，语速适中。

**适用对象**：已有基础词汇量（A2+），希望通过趣味内容提升听力和口语的学习者。

**为什么选老友记**：
- ✅ 发音清晰标准，语速适中
- ✅ 日常场景：咖啡馆、公寓、办公室
- ✅ 高频词汇占比高，学习效率好
- ✅ 文化宝库：90 年代美国生活全貌

**篇目一览**：

| 篇目 | 主题 |
|------|------|
| 第一篇·人物与背景 | 角色介绍与 90 年代纽约 |
| 第二篇·第一季经典 | Pilot、求职、约会、室友 |
| 第三篇·爱情与友情 | Ross&Rachel、Monica&Chandler |
| 第四篇·工作与人生 | 职业、金钱、公寓、纽约客 |
| 第五篇·笑料与文化 | 各角色幽默风格、节日特辑 |
| 第六篇·成长与告别 | 十年回顾、婚礼、生育、告别 |

- 🎯 热点话题：跨文化、LGBTQ+、科技融入生活

## 🗺️ 推荐学习路线

### 方案一：循序渐进（推荐，最系统）

```
第一阶段 ── 华尔街英语 第1-2篇（Pre-A1 → A1）
    ↓ 打好语法和基础词汇
第二阶段 ── 走遍美国（A1 → B1）
    ↓ 在生活场景中巩固和应用
第三阶段 ── 老友记（A2 → B2）
    ↓ 通过趣味内容提升听力和语感
第四阶段 ── 华尔街英语 第3-6篇（A2 → C1）
    ↓ 系统进阶到高级表达
第五阶段 ── 摩登家庭（A2 → B2）
    ↓ 接触当代口语和文化，融会贯通
```

### 方案二：兴趣驱动（适合有基础，A2+）

```
老友记 ──→ 摩登家庭 ──→ 走遍美国 ──→ 华尔街英语
       （趣味先导 → 当代口语 → 经典巩固 → 系统提升）
```

### 方案三：混合并行（适合时间充裕）

| 时间 | 内容 |
|------|------|
| 每周二、四 | 华尔街英语（系统学习） |
| 每周一、三 | 走遍美国（场景对话） |
| 周末 | 老友记或摩登家庭（泛听+跟读） |

## 💡 高效学习法

### 每课标准流程

```
1️⃣ 词汇预热 ── 浏览核心词汇，听发音，跟读（5分钟）
2️⃣ 剧情/对话 ── 通读中英对照对白，理解场景（10分钟）
3️⃣ 模仿跟读 ── 逐句跟读，注意连读、弱读、语调（10分钟）
4️⃣ 语法消化 ── 理解语法点，完成语法练习（10分钟）
5️⃣ 口语输出 ── 模拟场景，自问自答或录音回听（10分钟）
6️⃣ 练习巩固 ── 完成单元练习，自测掌握程度（5分钟）
                                   总约 50 分钟
```

### 黄金法则

- **跟读 > 默读**：英语是声音的语言，大声朗读比默读有效 3 倍
- **输入输出平衡**：每学 1 小时，至少做 15 分钟口语或写作输出
- **重复 > 贪多**：一个单元学三遍，胜过三个单元各学一遍
- **情境记忆**：在句子和对话中记单词，不要背孤立的单词表
- **影子跟读法**：播放音频，延迟 0.5-1 秒跟读，训练听力和发音同步

### 学习工具建议

| 用途 | 推荐工具 |
|------|---------|
| 词典 | 欧路词典 / Merriam-Webster |
| 查 IPA 音标 | 有道词典 / Cambridge Dictionary |
| 录音对比 | 手机录音机，跟读后回听对比 |
| 笔记整理 | Obsidian（本教程即为 Obsidian 友好格式） |
