# 摩登家庭自学教程

> 基于经典美剧 *Modern Family*（摩登家庭）的自学英语教程

## 关于本书

《摩登家庭》（*Modern Family*, 2009-2020）是一部获得过 22 项艾美奖的经典美式情景喜剧，以伪纪录片（mockumentary）的形式记录了普里切特（Pritchett）、邓菲（Dunphy）和塔克（Tucker）三个家庭的真实生活。

与传统的英语教材不同，摩登家庭中的对话**贴近当代美国人的真实口语**，涵盖了：
- **地道美语口语**：俚语、缩略语、日常对话
- **各种口音**：标准美式、哥伦比亚口音、不同社会阶层的语言
- **多代际沟通**：从老人到小孩，不同年龄段的表达方式
- **丰富场景**：家庭、学校、职场、旅行、节日等
