# 附录A：美语俚语与口语表达大全

> 本附录整理了《老友记》（Friends）全十季中出现的高频美语俚语与口语表达，按字母顺序排列。每个条目包含俚语/表达、中文解释、字面意思、使用场景说明及剧中例句，帮助读者在语境中理解和掌握地道美语口语。## A

### 1. Aww / 啊～

| 项目 | 内容 |
|------|------|
| **俚语/表达** | Aww |
| **中文解释** | 表感动、怜爱、惋惜或同情的感叹词 |
| **字面意思** | 无具体字面意思，为感叹音 |
| **使用场景说明** | 看到可爱的事物、听到感人的话或表示同情时使用 |
| **剧中例句** | *"Aww, look at the little duck!"* — 看到小鸭子时众人感叹 |

### 2. Absolutely / 绝对地、当然

| 项目 | 内容 |
|------|------|
| **俚语/表达** | Absolutely |
| **中文解释** | 表示强烈肯定，相当于"当然""绝对没错" |
| **字面意思** | 绝对地 |
| **使用场景说明** | 对某事表示百分之百的同意或确认，语气比"yes"更强 |
| **剧中例句** | *Monica: "Are you sure we can afford this?" Chandler: "Absolutely!"* (S01E06) |

### 3. Ass / 屁股、混蛋

| 项目 | 内容 |
|------|------|
| **俚语/表达** | Ass（如 get your ass over here / smart ass / pain in the ass） |
| **中文解释** | 屁股；也可用于构成复合词表示"讨厌的人/事" |
| **字面意思** | 驴；屁股 |
| **使用场景说明** | 非正式用语，表示催促、"聪明鬼"（带贬义）或"麻烦事" |
| **剧中例句** | *Chandler: "Stop being such a smart ass."* — 让Joey别耍贫嘴 |
