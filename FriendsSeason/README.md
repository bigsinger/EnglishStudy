# 老友记自学教程

> 基于经典美剧 *Friends*（老友记，1994-2004）的自学英语教程

## 关于本书

《老友记》（*Friends*）是史上最受欢迎的情景喜剧之一，播出了10季236集，获得62项艾美奖提名。数十年来，它一直是最受英语学习者欢迎的教材——**因为它的语言真实、幽默、贴近生活**。

### 为什么选老友记学英语？

- **清晰的发音**：六位主角的发音标准清晰，语速适中
- **日常场景**：咖啡馆闲谈、公寓聊天、办公室趣事——都是真实生活场景
- **丰富口音**：标准美式 + 英式（Ross的前妻）、各有所长的语言风格
- **高频词汇**：90%的对话由最常用的2000个单词构成
- **文化宝库**：90年代美国年轻人的生活方式、恋爱观、职场文化
