# 第一篇 · Pre-A1 入门篇：零基础起步

> 对应华尔街英语 Level 1-2 | CEFR Pre-A1

## 篇导读

欢迎来到英语学习的第一步！本篇是专为零基础学习者设计的入门课程，涵盖字母发音、基本问候、自我介绍和基础词汇。

## 学习目标

- ✅ 掌握26个英文字母的标准发音
- ✅ 学会基本的问候和告别用语
- ✅ 能用英语进行简单的自我介绍
- ✅ 掌握数字1-100、基础颜色和日常物品词汇

## 学习建议

- 大声跟读所有单词和句子
- 每天至少练习15分钟发音
- 完成每单元后的自测练习
