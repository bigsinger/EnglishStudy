# 1.4 数字、颜色与物品 (Numbers, Colors & Objects)

## 学习目标 Learning Objectives

- 能用英语数数1到20
- 能说出8种基本颜色的名称
- 能描述日常物品的颜色
- 能使用 "What is this?" 和 "This is..." 句型

## 实用句型 Useful Sentences

### 1. 数字 1-20 (Numbers 1-20)

| 数字 | 英文 | 数字 | 英文 | 数字 | 英文 | 数字 | 英文 |
|------|------|------|------|------|------|------|------|
| 1 | one | 6 | six | 11 | eleven | 16 | sixteen |
| 2 | two | 7 | seven | 12 | twelve | 17 | seventeen |
| 3 | three | 8 | eight | 13 | thirteen | 18 | eighteen |
| 4 | four | 9 | nine | 14 | fourteen | 19 | nineteen |
| 5 | five | 10 | ten | 15 | fifteen | 20 | twenty |

**数字规律总结**：
- 1-12：每个单词各不相同，需要单独记忆
- 13-19：以 **-teen** 结尾（thirteen, fourteen, fifteen...）
- 20：以 **-ty** 结尾（twenty）

> **小技巧**：13-19大部分是在个位数后加 teen，注意 three→thirteen, five→fifteen 有少量拼写变化。

### 2. 表达数字

- **How many...?** — 有多少……？
  - How many books? 有多少本书？
  - **One book.** 一本书。
  - **Two pens.** 两支笔。

- **There is / There are...** — 有……
  - There is **one** apple. 有一个苹果。
  - There are **three** cats. 有三只猫。

> **注意**：后面跟单数名词用 There is，跟复数名词用 There are。

### 3. 谈论颜色 Talking about Colors

- **What color is it?** — 它是什么颜色？
  - It is **red**. 它是红色。
  - It is **blue** and **white**. 它是蓝色和白色。

- **This/That is + 颜色** — 这/那是……颜色
  - This is a **red** apple. 这是一个红苹果。
  - That is a **black** cat. 那是一只是黑猫。

### 4. 基本问句——确认物品

- **What is this?** — 这是什么？
  - This is a **book**. 这是一本书。
  - This is a **pen**. 这是一支笔。

- **Is this a...?** — 这是一个……吗？
  - Is this a **table**? 这是一张桌子吗？
  - Yes, it is. 是的。
  - No, it is not. It is a **chair**. 不，不是。这是一把椅子。

## 语法要点 Grammar Point

### 1. 可数名词的单数和复数

英语中的可数名词有**单数**和**复数**两种形式。

**单数 → 复数的一般规则**：

| 规则 | 单数 | 复数 | 发音 |
|------|------|------|------|
| 一般加 -s | book | book**s** | /s/ |
| 一般加 -s | pen | pen**s** | /z/ |
| 一般加 -s | cat | cat**s** | /s/ |
| 以 s/x/sh/ch 结尾加 -es | box | box**es** | /ɪz/ |
| 以辅音+y结尾变y为i加-es | baby | bab**ies** | /iz/ |

**常用复数举例**：
- one book → two books
- one pen → five pens
- one cat → three cats
- one chair → four chairs
- one table → six tables

### 2. 指示代词 this / that

| 代词 | 含义 | 用法 | 例句 |
|------|------|------|------|
| **this** | 这个 | 指离说话人近的事物 | **This** is my book. |
| **that** | 那个 | 指离说话人远的事物 | **That** is a chair. |

> **记忆口诀**：this 近，that 远。this 是这个，that 是那个。

### 3. 形容词的位置

在英语中，**颜色形容词**放在名词的**前面**：

- a **red** apple（√）
- ~~apple red~~（✗）

- a **blue** pen（√）
- ~~pen blue~~（✗）

## 对话练习 Conversation Practice

**对话 1: 数字和物品**

- **A:** How many chairs? 有多少把椅子？
- **B:** One, two, three, four. Four chairs! 一、二、三、四。四把椅子！
- **A:** How many books? 有多少本书？
- **B:** One, two, three... ten books! 一、二、三……十本书！

**对话 2: 颜色问答**

- **A:** What color is the apple? 苹果是什么颜色？
- **B:** It is red. 它是红色。
- **A:** What color is the sky? 天空是什么颜色？
- **B:** It is blue. 它是蓝色。
- **A:** What color is this pen? 这支笔是什么颜色？
- **B:** It is black and white. 它是黑色和白色。

**对话 3: 教室里**

- **A:** What is this? 这是什么？
- **B:** This is a table. 这是一张桌子。
- **A:** What is that? 那是什么？
- **B:** That is a chair. 那是一把椅子。
- **A:** Is this your pen? 这是你的钢笔吗？
- **B:** Yes, it is my pen. It is blue. 是的，这是我的钢笔。它是蓝色的。

## 自测练习 Self-Test

1. 写出下列数字的英文：
   - 5 → \_\_\_\_\_\_\_
   - 8 → \_\_\_\_\_\_\_
   - 12 → \_\_\_\_\_\_\_
   - 20 → \_\_\_\_\_\_\_

2. 选择正确的单词填空：
   - The sky is \_\_\_\_\_\_\_ (blue / red).
   - Grass (草) is \_\_\_\_\_\_\_ (black / green).
   - Snow (雪) is \_\_\_\_\_\_\_ (white / yellow).

3. 将下列名词改为复数形式：
   - one pen → two \_\_\_\_\_\_\_\_
   - one book → three \_\_\_\_\_\_\_\_
   - one cat → five \_\_\_\_\_\_\_\_

4. 用 this 或 that 填空：
   - \_\_\_\_\_\_\_ (近处的) is my book.
   - \_\_\_\_\_\_\_ (远处的) is a chair.

5. 排列顺序，组成正确句子：
   - is / this / what / ?
   - red / a / is / apple / this

6. 将中文翻译成英文：
   - 有多少支钢笔？ \_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_
   - 这本书是什么颜色的？ \_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_**答案：**
1. five → eight → twelve → twenty
2. blue → green → white
3. pens → books → cats
4. This (近处) → That (远处)
5. What is this? → This is a red apple.
6. How many pens? → What color is this book?
