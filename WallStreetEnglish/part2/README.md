# 第二篇 · A1 基础篇：日常英语入门

> 对应华尔街英语 Level 3-5 | CEFR A1

## 篇导读

你已经掌握了基础问候和自我介绍，现在开始学习更多日常生活英语！本篇涵盖高频生活词汇、现在时态、家庭话题、时间表达和简单问答。

## 学习目标

- ✅ 掌握200+日常生活高频词汇
- ✅ 掌握一般现在时的肯定、否定和疑问句式
- ✅ 学会介绍家庭和朋友
- ✅ 能谈论时间、日期和天气
- ✅ 学会提出简单问题和请求
