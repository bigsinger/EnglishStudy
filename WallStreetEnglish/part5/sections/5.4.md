# 5.4 被动语态与复杂句 (Passive Voice & Complex Sentences)

## 学习目标 Learning Objectives

- 掌握被动语态在各种时态中的构成和用法
- 能正确使用定语从句和名词性从句
- 能在正式写作中有意识地使用被动语态

## 实用句型 Useful Sentences

### 被动语态基本句型

| 句型 | 主动句 | 被动句 |
|------|--------|--------|
| 一般现在时 | The team writes the report. | The report **is written** by the team. |
| 一般过去时 | The client signed the contract. | The contract **was signed** by the client. |
| 将来时 | We will finish the project. | The project **will be finished** (by us). |
| 现在完成时 | They have approved the proposal. | The proposal **has been approved**. |
| 情态动词 | We must complete the task. | The task **must be completed**. |

### 复杂句常用句型

| 句型 | 中文 | 例句 |
|------|------|------|
| The fact that… | ……的事实 | **The fact that** the contract was signed late caused problems. |
| It is believed that… | 据信…… | **It is believed that** the data was manipulated. |
| What matters is… | 重要的是…… | **What matters is** that the deadline was met. |
| The reason why… | ……的原因是 | **The reason why** the project failed **is that** the budget was insufficient. |

## 语法要点 Grammar Point

### 一、被动语态详解

**被动语态的构成：** be（根据时态变化）+ 过去分词 (past participle)

**什么时候使用被动语态？**

1. **动作的执行者未知或不重要**
   - My wallet **was stolen**. （不知道谁偷的）
   - The building **was built** in 1990. （谁建的并不重要）

2. **强调动作的承受者，而非执行者**
   - The client **was informed** of the delay. （强调客户，而不是谁通知的）
   - The report **has been submitted**. （强调报告本身）

3. **正式写作（学术、科技、商务）**
   - The experiment **was conducted** under controlled conditions.
   - A new strategy **will be developed** next quarter.

**被动语态各时态表格：**

| 时态 | 结构 | 例句 |
|------|------|------|
| 一般现在时 | am/is/are + done | English **is spoken** worldwide. |
| 现在进行时 | am/is/are + being + done | The proposal **is being reviewed**. |
| 现在完成时 | have/has + been + done | The contract **has been signed**. |
| 一般过去时 | was/were + done | The memo **was sent** yesterday. |
| 过去进行时 | was/were + being + done | The building **was being renovated**. |
| 过去完成时 | had + been + done | The presentation **had been prepared** before the meeting. |
| 一般将来时 | will + be + done | The report **will be submitted** on Friday. |
| 将来完成时 | will + have + been + done | By next month, the project **will have been completed**. |
| 情态动词 | modal + be + done | This **must be done** carefully. |

**主动句转被动句步骤：**

1. 找到原句的**宾语** → 变为主语
2. 动词改为 **be + 过去分词**，be 的时态与原来一致
3. 原主语 → **by + 主语**（可省略）

> 示例：The marketing team (S) launched (V) the campaign (O).
> → The campaign (S) was launched (V) by the marketing team.

### 二、定语从句 (Relative Clauses)

**定义：** 用来修饰名词的从句，相当于一个形容词的作用。

| 关系代词 | 用法 | 例句 |
|---------|------|------|
| who | 指人（作主语） | The client **who called yesterday** wants a refund. |
| whom | 指人（作宾语） | The manager **whom I met** gave me feedback. |
| which | 指物（作主语/宾语） | The contract **which was signed** is confidential. |
| that | 指人或物（口语常用） | The report **that I submitted** was approved. |
| whose | 表示所属关系 | The company **whose revenue declined** is restructuring. |

**非限制性定语从句**（用逗号隔开，补充说明）：

- The proposal, **which was submitted last week**, has been approved.
- The CEO, **who founded the company in 2010**, is stepping down.

### 三、名词性从句 (Noun Clauses)

**定义：** 在整个句子中充当名词角色的从句，可以做主语、宾语或表语。

| 功能 | 结构 | 例句 |
|------|------|------|
| 主语从句 | That / What + 完整句子 | **That the project failed** is obvious. / **What he said** is important. |
| 宾语从句 | that / whether / if + 句子 | I believe **that the strategy will work**. |
| 表语从句 | that + 句子 | The problem is **that we have no budget**. |
| 同位语从句 | that + 句子 | The fact **that she was promoted** surprised everyone. |

> **注意区分：** 定语从句的 that 是关系代词（作成分），名词性从句的 that 是连词（不作成分）。例：
> - 定语从句：The report **that I read** was helpful. （that 作 read 的宾语）
> - 名词性从句：The fact **that she left** is unfortunate. （that 不作成分）

## 对话练习 Conversation Practice

**场景：研究生讨论实验报告**

- **Professor Chen:** Let's discuss your lab report. I noticed that the passive voice **was used** throughout, which is appropriate for scientific writing.

- **Student:** Thank you. I learned that **it is believed that** passive **is preferred** in academic contexts.

- **Professor Chen:** Exactly. However, I noticed a few areas that need improvement. The introduction, **which was written** first, contains some **active** sentences that should **be transformed** to passive.

- **Student:** I see. For example, "I conducted the experiment" should **be changed** to "the experiment **was conducted**"?

- **Professor Chen:** Correct. In **scientific** writing, we want to **emphasize** the procedure, not the researcher. The **subject** should be the experiment, not you.

- **Student:** **What I don't understand** is when to use passive versus active in the discussion section.

- **Professor Chen:** Good question. The conclusion, **which** is the most important part, can use both. **The reason why** we mix them is **that** we need clarity and variety.

- **Student:** So the analysis **has been done** mostly correctly, but the methods section needs revision?

- **Professor Chen:** Yes. The methods section **must be written** entirely in passive voice. Everything **has been designed** to highlight the scientific process, not the person.

> **注意观察**：对话中使用了多种时态的被动语态（was used, was written, should be transformed, was conducted, has been done, must be written, has been designed），以及定语从句（which was written first, which is the most important part）、主语从句（What I don't understand is…）和表语从句（The reason why…is that…）。
