# 5.3 条件句与虚拟语气 (Conditionals & Subjunctive Mood)

## 学习目标 Learning Objectives

- 掌握四种条件句的结构和用法
- 能区分真实条件和非真实条件
- 能使用虚拟语气表达假设、愿望和委婉请求

## 实用句型 Useful Sentences

### 条件表达 (Expressing Conditions)

| 句型 | 中文 | 例句 |
|------|------|------|
| If… then… | 如果……那么…… | **If** we miss the deadline, **then** we lose the contract. |
| Unless… | 除非…… | **Unless** we negotiate better terms, the deal will fall through. |
| Provided that… | 只要…… | **Provided that** the client agrees, we can move forward. |
| Suppose… | 假设…… | **Suppose** we don't get the funding — what's our backup plan? |

### 假设与虚拟 (Hypotheticals)

| 句型 | 中文 | 例句 |
|------|------|------|
| If I were you… | 如果我是你…… | **If I were you**, I would accept the proposal. |
| If only… | 要是……就好了 | **If only** we had more time to prepare. |
| I wish… | 我希望…… | **I wish** we could collaborate more effectively. |
| It's time… | 是时候……了 | **It's time** we solved this problem. |

## 语法要点 Grammar Point

### 条件句的四种类型

英语中的条件句分为四种，从"很可能发生"到"绝不可能发生"。

**1. 零条件句 (Zero Conditional)**

**结构：** If + 现在时, 现在时

**用法：** 表示普遍真理、科学事实或习惯性结果（100% 肯定）

- **If** you heat ice, it **melts**.
- **If** the client is unhappy, we **offer** a refund.
- Water **boils** **if** you heat it to 100°C.

> 零条件句可以用 **when** 替代 **if**，意思不变：When you heat ice, it melts.

**2. 第一条件句 (First Conditional)**

**结构：** If + 现在时, will + 动词原形

**用法：** 表示将来很可能发生的真实条件（真实可能，≈50-90% 可能性）

- **If** we **negotiate** well, we **will get** a better contract.
- **If** the client **approves** the proposal, we **will start** next week.
- We **will miss** the deadline **unless** we **work** faster.

> **第一条件句的变体：** can / may / might 也可以代替 will：If we hurry, we **can** catch the train.

**3. 第二条件句 (Second Conditional)**

**结构：** If + 过去时, would + 动词原形

**用法：** 表示与现在事实相反或将来不太可能的假设（虚拟，≈10% 可能性）

- **If** I **were** you, I **would accept** the offer. （事实上我不可能是你）
- **If** we **had** more budget, we **would hire** more staff. （实际上我们没有更多预算）
- **If** the deadline **were** next Friday, we **would have** more time. （实际上截止日期就是这个周五）

> **注意：** 在第二条件句中，**were** 用于所有人称（I were, he were, she were），这是虚拟语气的标志。

**4. 第三条件句 (Third Conditional)**

**结构：** If + had + 过去分词, would have + 过去分词

**用法：** 表示与过去事实相反的假设（后悔、反思，0% 可能性，因为是过去的事）

- **If** we **had negotiated** harder, we **would have saved** more money. （但当时没有努力谈判）
- **If** she **had known** about the deadline, she **would have finished** on time. （但她当时不知道）
- **If** they **had collaborated** with us, the project **would have been** a success. （但他们没有合作）

### 条件句对比表

| 类型 | 结构 | 可能性 | 时间 | 例句 |
|------|------|--------|------|------|
| 零条件 | If + 现在, 现在 | 100% | 任何时间 | If it rains, the ground gets wet. |
| 第一条件 | If + 现在, will + 原形 | 50-90% | 将来 | If it rains, we will stay home. |
| 第二条件 | If + 过去, would + 原形 | ≈10% | 现在/将来 | If it rained, we would stay home. |
| 第三条件 | If + had done, would have done | 0% | 过去 | If it had rained, we would have stayed home. |

### 常用替代连接词

| 连接词 | 含义 | 用法 | 例句 |
|--------|------|------|------|
| unless | 除非……否则 | = if not | We'll fail **unless** we try harder. |
| provided / providing | 只要 | = if and only if | **Provided** you finish on time, you can leave early. |
| as long as | 只要 | = if | **As long as** it's within budget, it's fine. |
| even if | 即使 | 让步条件 | **Even if** it rains, we'll go. |
| in case | 以防 | 预防条件 | Take an umbrella **in case** it rains. |
| suppose / supposing | 假设 | = what if | **Suppose** we fail? |

## 对话练习 Conversation Practice

**场景：创业团队讨论融资方案**

- **Lisa:** OK team, let's talk about the investor meeting next week. **Suppose** the investors ask about our revenue model — what do we say?

- **Mark:** **If** our current growth rate **continues**, we **will reach** profitability by Q2. That's a real **possibility**.

- **Lisa:** Good. But what about **hypothetical** scenarios? **If** the market **were** to crash, what **would** happen?

- **Mark:** **If** that **happened**, we **would** need to cut costs immediately. **Provided that** we maintain good relationships with our clients, we could survive.

- **Lisa:** **Unless** they cancel their contracts, of course. Then we'd be in trouble.

- **Mark:** Exactly. And **if** we **had secured** more funding last year, we **would have been** in a much stronger position today.

- **Lisa:** Well, we can't change the past. But think about the **consequences** of our current strategy. **If** we **don't** close this deal, we **will have** to reconsider our approach.

- **Mark:** True. **If I were** the investors, I **would** want to see a clear path to revenue. Let's make sure our **presentation** is solid.

- **Lisa:** Agreed. **If only** we had more time to prepare! But we don't. So let's focus on the essentials.

> **注意观察**：对话中依次使用了第一条件句（If our current growth rate continues, we will reach…）、第二条件句（If the market were to crash… / If I were the investors…）、第三条件句（If we had secured more funding… we would have been…）和 If only 虚拟结构。
