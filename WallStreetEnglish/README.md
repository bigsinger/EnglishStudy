# 华尔街英语自学教程

> 基于 Wall Street English 20级课程体系的自学教材

## 关于本书

本书以华尔街英语（Wall Street English）20级课程体系为蓝本，按照欧洲语言共同参考框架（CEFR）标准，系统覆盖从 **Pre-A1（零基础）** 到 **C1（高级精通）** 六个等级。

全书共分为 **6 篇、28 个单元**，每单元包含：
- **核心词汇表**：10-15个关键词汇，标注CEFR等级，配英文释义和例句
- **实用句型**：5-10个高频句型，中英对照
- **语法要点**：语法点讲解与练习
- **对话练习**：真实场景对话
- **自测练习**：巩固所学内容

## 学习建议

- 每单元建议学习时间：1-2小时
- 先学词汇 → 再学句型 → 掌握语法 → 练习对话
- 大声朗读所有例句和对话
- 完成自测练习后进入下一单元
