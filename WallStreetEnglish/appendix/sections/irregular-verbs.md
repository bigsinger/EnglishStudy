| write | wrote | written | 写 |## 💡 学习建议：如何记忆不规则动词

不规则动词虽然看起来杂乱无章，但其实很多动词之间存在规律。以下分组记忆法可以帮助你高效掌握：

### 1. AAA型（三形相同）
**模式**：原形 = 过去式 = 过去分词
**示例**：cut → cut → cut, put → put → put, set → set → set, hit → hit → hit, let → let → let, shut → shut → shut, spread → spread → spread, cost → cost → cost, burst → burst → burst

### 2. ABB型（过去式 = 过去分词）
**模式**：过去式和过去分词相同
**示例**：
- **-ught/-aught 结尾**：buy → bought → bought, bring → brought → brought, catch → caught → caught, fight → fought → fought, think → thought → thought, teach → taught → taught, seek → sought → sought
- **-t/-d/-lt 结尾**：lend → lent → lent, send → sent → sent, build → built → built, spend → spent → spent, bend → bent → bent, feel → felt → felt, keep → kept → kept, leave → left → left, mean → meant → meant, sleep → slept → slept, sweep → swept → swept, weep → wept → wept, meet → met → met, lose → lost → lost, shoot → shot → shot, sit → sat → sat, stand → stood → stood, understand → understood → understood, have → had → had, make → made → made, pay → paid → paid, say → said → said

### 3. ABA型（过去分词 = 原形）
**模式**：过去分词和原形相同
**示例**：become → became → become, come → came → come, run → ran → run, overcome → overcame → overcome

### 4. ABC型（三者完全不同）
**模式**：原形、过去式、过去分词均不同

### 记忆技巧
1. **按模式分组**：不要逐个背，先按以上分组找出规律
2. **编小故事**：把你的分组变成一个短故事更容易记住
3. **高频优先**：先掌握最常用的50个不规则动词，再扩展
4. **主动使用**：每天用刚学的动词造3个句子（一般现在时/过去时/完成时各一句）
5. **多听多读**：在阅读和听力中留意不规则动词的使用场景
6. **制作闪卡**：正面写原形，背面写过去式和过去分词
