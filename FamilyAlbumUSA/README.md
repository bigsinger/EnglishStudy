# 走遍美国自学教程

> 基于经典美语教材 Family Album U.S.A.（走遍美国）的自学教程

## 关于本书

《走遍美国》（Family Album U.S.A.）是一套在全球35个国家同步使用的经典美语教材，由美国新闻总署赞助制作。本书以斯图尔特（Stewart）一家的日常生活为主线，通过真实的场景对话帮助学习者掌握地道美语。

全书分为 **6篇、29个单元**，每单元包含：
- **剧情概要 Story Summary**：了解本集故事
- **经典台词 Key Dialogues**：精选对白，中英对照
- **核心词汇 Core Vocabulary**：重点单词与短语
- **实用表达 Useful Expressions**：地道美语表达
- **文化注释 Cultural Notes**：了解美国文化
- **语法要点 Grammar Point**：语言点讲解
- **练习 Practice**：巩固所学
