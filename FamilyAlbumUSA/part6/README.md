# 第六篇·庆典与告别

> 难忘的夜晚、依依惜别、纽约之梦、惊喜与感动、乡村之旅——斯图尔特一家的故事在欢声笑语中画上完美的句号，也开启新的篇章。

## 本篇概览

| 单元 | 内容 | 重点 |
|------|------|------|
| 6.1 | A Night to Remember | 晚会节目、社交聚会、最高级 |
| 6.2 | Parting Friends | 送别用语、表达不舍、将来时 |
| 6.3 | The Big Apple | 纽约地标、城市生活、方向问路 |
| 6.4 | The Surprise | 制造惊喜、发现秘密、间接引语 |
| 6.5 | A Trip to the Country | 乡村生活、自然词汇、比较句型 |
| 6.6 | The End / A New Beginning | 告别与新起点、回顾展望、总结 |

## 学习目标

- 学会组织晚会和社交活动用语
- 掌握告别和送别的英语表达
- 了解纽约地标和城市文化
- 整体回顾全书语法与词汇
